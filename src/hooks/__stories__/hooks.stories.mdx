# Hooks

<Description
  markdown={dedent`
The \`useCoordinates()\` hook does three things:
1. request location permission from the user
2. returns the latitude and longitude retrieved (\`{latitude: number; longitude: number;}\`)
3. informs if location data is loading`}
/>

<Source
  language="jsx"
  dark
  code={dedent`
import {useCoordinates} from 'react-weather-component';
${' '}
const MyComponent = () => {
  const {coordinates, isLoading} = useCoordinates();
  
  return isLoading ? <div>Loading...</div> : (
    <div>
      <p style={styles.text}>isLoading: {\`\${isLoading}\`}</p>
      <p>latitude: {coordinates?.latitude}</p>
      <p>longitude: {coordinates?.longitude}</p>
    </div>
  );
}
`}
/>
