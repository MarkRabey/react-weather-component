# Example

You must register for an API key from [OpenWeather](https://openweathermap.org/api).

<Source
  dark
  code={dedent`
import WeatherComponent from 'react-weather-component';
${' '}
const App = () => {
  return (
    <WeatherComponent apiKey="YOUR_API_KEY" />
  );
}
${' '}
export default App;
`}
/>
